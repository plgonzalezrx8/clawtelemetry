See what your OpenClaw agents are actually doing. Token costs, live traces, memory browser, sub-agent activity -- all in one dashboard.

## Install

```bash
pip install clawmetry
```

## Run

```bash
clawmetry
```

Opens the dashboard at `http://localhost:4321`. Works with any existing OpenClaw setup -- no config needed.

## What you get

- **Live agent traces** -- every tool call, token usage, and sub-agent spawn in real time
- **Token cost tracking** -- see exactly what each session costs
- **Memory browser** -- inspect what your agents remember
- **Cron job monitor** -- status of all scheduled agent tasks
- **Sub-agent activity** -- full tree of spawned agents and their outcomes
